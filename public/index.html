One Platform Foundation

The Open Platform Foundation is a global, open-source initiative to help the Android ecosystem evolve by enabling developers to easily code and submit their apps across multiple alternative app stores. AppDF - App Description File

Uploading Android application to several appstores could be time consuming. AppDF format is designed to simplify this process. You describe your Android application once, create a simple AppDF archive that includes the XML description, APK file(s), screenshots, app icon, promo images, etc and just upload this AppDF file to all the stores that support AppDF uploading.

OpenIAB - Open In-App Billing

There are several different In-App Purchase APIs already and this number is increasing. We are creating an open source library that will wrap appstore in-app purchase APIs of all the stores and provide an easy way for Android developers to develop their apps/games in a way that one APK will work in all the stores and automatically use right in-app purchase API under each store. Plus we are developing an open in-app billing API that stores could implement to support all the built APK files using this library.

  • OpenIAB library is not ready for your app development yet
